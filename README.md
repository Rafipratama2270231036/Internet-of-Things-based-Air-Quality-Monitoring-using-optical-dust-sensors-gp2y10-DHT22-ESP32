# Internet-of-Things-based-Air-Quality-Monitoring-using-optical-dust-sensors-gp2y10-DHT22-ESP32
ESP32 Multi-Sensor Air Quality Monitoring – Bagian skripsi: Prediksi Kualitas Udara berbasis IoT dengan Time Series. Firmware ESP32 pengirim data PM2.5 (GP2Y1010AU0F), suhu &amp; kelembaban (DHT22) via UART. Dilengkapi auto-baseline, kalibrasi, running average, dan batasan 0–500 µg/m³.
